Monkeys with Typewriters is a satirical project. MWT-1 generates random text using a 150-token vocabulary on a $3 ESP8266 microcontroller. It is not artificial intelligence. It is not machine learning. It is a random number generator with a vocabulary list. Every metric on this page is technically true. Think about that the next time someone shows you an AI benchmark.
