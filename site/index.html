Built with irritation by Jay Neawedde

Monkeys with Typewriters is a satirical project. MWT-1 generates random text using a 150-token vocabulary on a $3 ESP8266 microcontroller. It is not artificial intelligence. It is not machine learning. It is a random number generator with a vocabulary list and 14 API endpoints. The multi-agent orchestration runs the same function multiple times. The RAG pipeline retrieves random words from an array. The chain-of-thought prepends phrases like "analyzing the implications of" to random text. The guardrails always pass because random buzzwords cannot be harmful. The embeddings are trigonometric hashes. The fine-tuning adds a word to an array. The tool use calls random(). The multimodal reads a voltage pin.

Every metric on this page is technically true. Every feature description is accurate. If any of this sounds like a product you're currently paying for, we encourage you to ask your vendor some pointed questions.
